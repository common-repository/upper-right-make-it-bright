== Description ==

This plugin enables a widget area above the header on most Genesis Themes. Note: This requires a Genesis Child Theme to work.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Place a widget in the Upper Right, Make It Bright widget area to active the sidebar.
